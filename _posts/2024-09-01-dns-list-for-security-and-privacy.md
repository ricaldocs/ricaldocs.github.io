## 1. Cloudflare
### Desktop

| IPv4                            | IPv6       				   |
| :--------------------- | :---------------------------- |
| 1.1.1.1                       | 2606:4700:4700::1111 |
| 1.0.0.1 			| 2606:4700:4700::1001 |

### Android
Cari pengaturan DNS (biasanya bernama `Private DNS` atau `DNS Pribadi`) dan masukkan informasi berikut:
```
security.cloudflare-dns.com
```
